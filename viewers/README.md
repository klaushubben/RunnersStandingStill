# Viewers

Website and display contexts for the project.

- `site-page/` - public Klausblocks project page, mint UI, style, local config, and render harness.
- `screen-viewer/` - full-screen display viewer with `view=screen` / `view=object` URL modes.

These are not the on-chain renderer source of truth. They are website/display wrappers around the current renderer artifact.
